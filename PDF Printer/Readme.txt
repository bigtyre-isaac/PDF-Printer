Big Tyre PDF Printer
-----------------------------------
This program prints a PDF document. 
It will use the default printer or you can specify the printer using the option -p or --printerName followed by the name of the printer.
The document will be printed using whatever the default settings are for that printer. It doesn"t prompt for any printer or document settings. 
It will print every page in the document using whatever orientation and sizing has been specified in the document.

Usage: PDFPrinter.exe (-p "Printer Name") -f "C:/filename.pdf"
